## WebSocket Chat Application

## Steps to Setup

```bash
cd chat-demo
mvn package
java -jar target/chat-demo-1.0.0.jar
```

Alternatively, you can run the app directly without packaging it like so -

```bash
mvn spring-boot:run
```
